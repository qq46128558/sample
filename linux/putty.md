# putty

windows下的遠程工具

## 命令行模式

	putty 用戶名@服務器地址
	putty 用戶名@服務器地址 -pw 密碼
	putty 服務器地址 -l 用戶名
	putty 服務器地址 -l 用戶名 -pw 密碼

~~~
-ssh 使用ssh模式
-P 指定端口
~~~
